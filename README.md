# Gerador de Documentos Avulsos de Rescisão

Ferramenta interna da Mury Assessoria Contábil: calcula verbas rescisórias e gera Acordo Rescisório / Recibo Avulso prontos para impressão.

A base de clientes **não** faz parte deste repositório — é importada localmente pelo usuário e fica apenas no navegador (localStorage).
